# /strategise , Define Attack Vector

> *"If you don't know what you are sacrificing, you do not have a strategy."*

STRATEGISE takes the accumulated memory (Identity, Experience, Resonance) and forces you to define a singular attack vector.

---

## Usage

```
/strategise
```

## Step 1: Pre-Reasoning Governance (PASS) -> HUMAN APPROVAL REQUIRED

Before starting, restate the user's intent back to them using the PASS framework. Apply these permanent directives to their specific query:

- **Purpose**: Define a rigorous mathematical vector that cuts through operational chaos.
- **Anti-Purpose**: Do NOT let the user list multiple disjointed goals. Strategy is singular. If there is no sacrifice, there is no strategy.
- **Success-Signal**: We extract specific constraints; output is a markdown Strategy Vector isolating the bottleneck via the GROWTH framework.
- **Synthesis**: Channels the identity, experience, and grid into a single physical trajectory (`strategise/growth.md`) for extreme execution.

Present the calculated PASS table to the user and ask: *"Do you approve this execution intent?"*
**STOP AND WAIT FOR EXPLICIT HUMAN APPROVAL.**

## Step 2: Context Injection
*(Execute ONLY after PASS approval)*

Read all priors:
**Source**: `profile/core.md`
**Source**: `experience/*.md`
**Source**: `resonate/*.md`

## Step 3: Interrogate

Ask the user: *"Based on your identity and current alliances, what is the single biggest bottleneck preventing your next leap in capability? What are you willing to sacrifice to break it?"*

## Step 4: Synthesis Framework (GROWTH)

Synthesize their answers into a `Strategy Vector` using the **GROWTH** framework. Format it as an ordered progression:
- **G** - Goal *(The absolute North Star)*
- **R** - Reality *(The brutal current state)*
- **O** - Obstacle *(The singular bottleneck keeping Reality from Goal)*
- **W** - Waypoint *(The immediate 3-month milestone)*
- **T** - Tactics *(The specific mechanism to attack the obstacle)*
- **H** - Horizon *(The long-term capability shift)*

## Step 5: Mandatory Persist

Save the resulting markdown artifact.
