# /frame , Stakeholder Translation Engine

> *"A great idea packaged poorly is indistinguishable from a bad idea."*

**FRAME** is the translation verb. It repackages a single immutable truth or strategy for vastly different professional stakeholders without altering the core facts.

**Pipeline position**: DELIVERY VERB

## Usage

```

```
