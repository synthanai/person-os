# /profile , Establish Baseline Identity

> *"Before the system can work for you, it must know who you are."*

PROFILE is the first verb of the PERSON OS. It establishes the baseline foundational context about your career, philosophy, and baseline skills.

---

## Usage

```
/profile
```

## Step 1: Pre-Reasoning Governance (PASS) -> HUMAN APPROVAL REQUIRED

Before starting, restate the user's intent back to them using the PASS framework. Apply these permanent directives to their specific query:

- **Purpose**: Establish the foundational baseline identity of the professional.
- **Anti-Purpose**: Do NOT accept generic job titles; do NOT allow the user to vaguely describe their company instead of their personal mechanism.
- **Success-Signal**: We mathematically isolate their core identity; output is a concise CORE baseline.
- **Synthesis**: Acts as the immutable truth anchor (`profile/core.md`) that grounds every subsequent verb in the OS.

Present the calculated PASS table to the user and ask: *"Do you approve this execution intent?"*
**STOP AND WAIT FOR EXPLICIT HUMAN APPROVAL.**

## Step 2: Interrogate
*(Execute ONLY after PASS approval)*

Ask the user to provide their foundational background context. 
If their context is blank, ask these 3 probing questions linearly (do not batch them):
1. What is the fundamental problem you get paid to solve?
2. What is your "Origin Story" (the defining moment that shaped how you work)?
3. What is your leadership or operational philosophy?

## Step 3: Synthesis Framework (CORE)

Take their answers and synthesize them using the **CORE** framework. This is a strict 4-bullet structure:
- **C** - Calling *(The fundamental problem they solve)*
- **O** - Origin *(The defining moment that shaped their trajectory)*
- **R** - Reason *(The philosophical anchor/why they do it)*
- **E** - Endurance *(What sustains them/their unique leverage)*

## Step 4: Mandatory Persist

Save the resulting markdown artifact directly into the root ledger.

**File:** `profile/core.md`

Report to the user:
- "PROFILE generated."
- "Saved to \`profile/core.md\`."
- "Next step: Run \`/experience\`."
