# OSS / IP BOUNDARY

This repository is **open source** under MIT License.

**What is open:**
- All GPT instruction files (`instructions.md`)
- All question files (`*_questions_*.md`)
- All output templates (`*_output_template.md`)
- Workshop materials
- Documentation

**What is NOT in this repository:**
- The SYNTHAI MBS framework internals (proprietary — see synthai-philosophy repo)
- ARIVAR profiling engine (proprietary)
- CORE extraction protocols (proprietary)
- Profile Tesseract schema (proprietary)

The open-source kit is a standalone product. It does not require access to any proprietary SYNTHAI systems to function.
